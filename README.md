## Ecommerce API project

Back-end REST API built with Laravel.

## Tech Stack

- PHP 8+
- Laravel
- MySQL

## Installation

1. Clone repository.
2. Run composer install.
3. Configure .env file.
4. Run 'php artisan migrate'.
5. Run 'php artisan serve'.
